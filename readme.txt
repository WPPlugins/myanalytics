== Description ==

Un simple plugin qui permet de mettre le tag de Google Analytics et qui respecte les normes de la CNIL.
Il affiche ainsi une barre pour avertir l'utilisateur de la présence de cookie.
Il propose l'acceptation et le refus des cookies de Google Analytics, et il gère également le DNT (DoNotTrack).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/myanalytics` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MyAnalytics screen to configure the plugin
4. Enter the Google Analytics's code of your website
